Substrand Counter

ID: Finding Common Motif
Time Complexity: Linear
Input-Type:Two Plain Text Sequence Strings
Input: Sequence String and Motif String
Output: Indices of Sequence String


The motif algorithm coded within the web application is an algorithm which searches for substrings within a strand inputted by the user. The text file that is uploaded must have the strand on the first line with the desired substring on the second. Upon taking the strand, the algorithm searches for the ordered character at which the substring occurs, if applicable. The numbered characters where the substring is found is then outputted.


This is useful in genome analysis and bioinformatics as repetitive sequences within a DNA strand can be recorded and can be used in the identification of potential trinucleotide repeat disorders, a particularly harmful class of DNA motifs that can appear in non-coding regions of sequences, which is the causing factor behind disorders such as Huntington’s disease. Trinucleotide repeat disorders are the results of repetitive sequences that mostly impact the nervous system. These disorders are genetically inherited as each generation has more repetitive sequences present in DNA (Ramakrishnan & Gupta, 2022).

data = open('data.txt','r').read().split('\n')
strand = data[0]
ans =''
n=len(data[1])
for i in range(len(strand)):
	if data[1] == strand[i:i+n]:
		if ans == '':
			ans += str(i+1)
		else: 
			ans += ' '+str(i+1)
print(ans)

The code functions by storing the both the sequence and motif substring. A for loop iterates through the sequence string, comparing the substring to the consecutive characters of the string of the substring length, and recording all the indices of the sequence string that matched.
